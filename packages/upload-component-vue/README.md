# @chunkflow/upload-component-vue

Ready-to-use Vue UI components for ChunkFlow Upload SDK.

## Installation

```bash
pnpm add @chunkflow/upload-component-vue
```

## Components

- `UploadButton` - File selection button
- `UploadProgress` - Progress indicator
- `UploadList` - Upload task list
- `UploadDropzone` - Drag and drop zone
